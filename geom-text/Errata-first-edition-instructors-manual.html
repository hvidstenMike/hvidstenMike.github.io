Errata (Instructor's Manual)

First Edition, First Printing

This web page lists errors that have been discovered in the first edition of the Instructor's Resource Manual to Accompany Geometry with Geometry Explorer, by Mike Hvidsten, published by McGraw-Hill, January 2005.  Please note that any changes involving mathematical notation are expressed in TeX form.
If you discover errors other than those listed, please email them to hvidsten(at)gustavus.edu
