Latest Version:  Geometry Explorer Version 3.0  (Released 8/03/2015)  To download any of these installer files, right click (ctrl or apple-click for one button mice) on the file and choose "Save Link As" to save the installer file to your computer. Then, on Macintosh and Windows machines, double-click the installer file to run the installation program.  On Linux machines follow the instructions below.

    Windows:  Minimum Required configuration:  Windows 7 or above. CPU speed >= 500 mhz, at least 140 mb of hard disk space (~60 mb for the installer and 74 mb for the program once installed), and 1 gb of RAM.  A Java virtual machine (JVM) is required at the level of JDK 1.7 or above. (You can download the latest version of Java from https://www.java.com/en/download/manual.jsp)

      Download Geometry Explorer version 3.0 (8/03/2015)  for Windows  (60.42 mb). Once the installer file is downloaded, right-click on the file and choose "Run as Administrator" to run the installer. 

    Linux:  Minimum Required configuration:  CPU speed >= 500mhz, at least 110 mb of hard disk space (~36 mb for the compressed distribution and 74 mb for the program files once decompressed), and 1 gb of RAM.  A Java virtual machine (JVM) is required at the level of JDK1.7 or above. (You can download the latest version of Java from https://www.java.com/en/download/manual.jsp). Download the following file and install the program by typing "sh GEX_unix_3_0.sh" in a shell.  After installation, you may need to change permissions on the executable (labeled "GeometryExplorer"). 

      Download Geometry Explorer version 3.0 (8/03/2015) for Linux (35.80 mb)
Macintosh OS X :  Minimum Required configuration:  OS X 10.7.3 or above.  CPU speed >= 500mhz, at least 110 mb of hard disk space (~36 mb for the compressed distribution and 74 mb for the program files once decompressed), and 1 gb of RAM. A Java virtual machine is required at the level of Java 8 or above. (You can download the latest version of Java from https://www.java.com/en/download/manual.jsp)
