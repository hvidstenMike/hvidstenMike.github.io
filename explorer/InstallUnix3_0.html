Instructions for Installing Geometry Explorer on Unix (or Unix-like) Computers

Step 1: Download the file gex_unix_3_0.tar.gz from the Geometry Explorer download page. 

Step 2: Un-Gzip and untar the file, creating the GeometryExplorer directory.

Step 3: After the installation has finished, the program may be run by typing "./GeometryExplorer" in the installation directory. You do not need to use a "java" or "jre" command to start the program.
